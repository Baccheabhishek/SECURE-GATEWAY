# Secure Gateway

An AI-powered API Security Gateway built with Node.js, Express, and MongoDB.

## Features

- User Registration
- JWT Authentication (In Progress)
- Role-Based Access Control (In Progres)
- Rule-Based Threat Detection (Planned)
- AI Threat Detection (Planned)

## Tech Stack

- Node.js
- Express
- MongoDB
- Mongoose
- JWT
- bcrypt
- Ollama (Upcoming)
